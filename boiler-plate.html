Comment commencer une nouvelle page html

On pourrait simplement taper à main le code qu'on trouve en commun dans toutes les pages html. On peut coller une copie de cette code en copiant d'un fichier texte qu'on garde quelque part. Des texteurs, des logiciels de texte spécialisés pour écrire du html, peuvent avoir une fonction pour générer cette code.

  • BBEDit: CtrlCmdN
  • Visual Studio Code: !tab

Voici le 'boilerplate' code que BBEdit génère: 

  	<!DOCTYPE html>
	<html lang="en">
	<head>
		<meta charset="utf-8"/>
		<title>BBEdit boiler plate</title>
		<meta name="generator" content="BBEdit 13.5"/>
	</head>
	<body>

	</body>
	</html>
